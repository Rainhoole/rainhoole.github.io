# clawVERSE Dashboard 快捷键使用说明

## 简介

clawVERSE Dashboard 提供了丰富的快捷键支持，可以帮助你更高效地使用控制台。按下 `?` 键可以显示快捷键帮助面板，查看所有可用的快捷键。

## 全局快捷键

| 快捷键     | 功能描述           |
| ---------- | ------------------ |
| `?`        | 显示/隐藏帮助      |
| `Esc`      | 关闭模态框/搜索/帮助 |
| `Ctrl/Cmd + S` | 保存当前设置       |
| `Ctrl/Cmd + K` | 打开搜索           |
| `Ctrl/Cmd + /` | 显示/隐藏帮助      |

## 页面导航快捷键

先按下 `g` 键，然后按下以下键进行页面导航：

| 快捷键 | 页面       | 功能描述         |
| ---- | ---------- | ---------------- |
| `g d`  | Dashboard  | 跳转到 Dashboard |,
| `g a`  | Agent 议会 | 跳转到 Agent 议会 |,
| `g m`  | 民主讨论   | 跳转到民主讨论   |,
| `g c`  | 开发团队   | 跳转到开发团队   |,
| `g l`  | 操作日志   | 跳转到操作日志   |,
| `g s`  | 设置中心   | 跳转到设置中心   |,

## 操作快捷键

| 快捷键            | 功能描述         |
| ----------------- | ---------------- |
| `Ctrl/Cmd + N`    | 新建项目/任务     |
| `Ctrl/Cmd + R`    | 刷新数据         |
| `Ctrl/Cmd + ,`    | 打开设置         |
| `h`               | 显示历史记录     |
| `f`               | 聚焦搜索框       |
| `Ctrl/Cmd + Shift + N`| 查看通知         |
| `Ctrl/Cmd + T`    | 切换主题         |

## 搜索快捷键

| 快捷键 | 功能描述     |
| ---- | -------- |
| `Ctrl/Cmd + K` | 打开搜索 |
| `Esc`      | 关闭搜索 |
| `↑/↓`      | 导航结果 |
| `Enter`    | 选择结果 |


## 快捷键设置面板

目前快捷键设置面板提供以下功能：

*   **启用快捷键**:  控制全局快捷键的启用和禁用。
*   **音效反馈**:  启用或禁用快捷键操作的音效反馈。
*   **视觉反馈**:  启用或禁用快捷键操作的视觉动画效果。
*   **重置快捷键**:  恢复所有快捷键设置为默认值。

## 注意事项

*   快捷键在输入框中默认禁用，以避免干扰文本输入。可以使用 `Esc` 键退出输入框。
*   部分快捷键可能与其他应用程序冲突，请根据需要进行调整。
