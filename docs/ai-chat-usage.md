# AI Chat 使用文档

## 简介

AI Chat 模块为 clawVERSE Dashboard 提供 AI 助手功能，允许用户通过自然语言与系统交互，并执行命令。

## 功能

- **AI 助手聊天窗口**：提供与 AI 助手对话的界面。
- **自然语言查询**：用户可以使用自然语言描述需求，AI 助手将尝试理解并给出相应的响应。
- **智能建议**：AI 助手会根据当前上下文提供智能建议，帮助用户快速找到所需功能。
- **命令执行**：用户可以使用特定命令执行系统操作。

## 使用方法

1.  **打开 AI 助手**：点击右下角的 AI 聊天按钮，打开 AI 助手聊天窗口。
2.  **发送消息**：在输入框中输入消息，然后点击发送按钮或按 Enter 键发送消息。
3.  **使用命令**：在输入框中输入 `/` 开头的命令，然后按 Enter 键执行命令。
4.  **查看建议**：点击 AI 助手提供的智能建议，快速发起查询。

## 命令列表

| 命令      | 描述               |
| ----------- | ------------------ |
| `/help`    | 显示帮助信息         |
| `/status`  | 显示系统状态         |
| `/search`  | 执行搜索             |
| `/export`  | 导出数据             |
| `/theme`   | 切换主题             |
| `/stats`   | 显示统计信息         |
| `/clear`   | 清空对话             |
| `/navigate` | 导航到指定页面       |

## 注意事项

-   AI 助手仍在开发中，部分功能可能无法正常使用。
-   请使用清晰、简洁的语言描述你的需求。
-   如果遇到问题，请联系开发团队。
